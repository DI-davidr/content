
#### Scripts
##### New: CortexXDRCloudProviderWidget
- This script returns an HTML result of the cloud providers in the incident. The result is centered, colored with a large font (Available from Cortex XSOAR 6.0.0).
##### New: CortexXDRAdditionalAlertInformationWidget
- This script give additional original alert information from the context (Available from Cortex XSOAR 6.0.0).
##### New: CortexXDRIdentityInformationWidget
- Widget to show cortex XDR identity information (Available from Cortex XSOAR 6.0.0).
##### New: CortexXDRRemediationActionsWidget
- Widget to show Cortex XDR remediation actions information (Available from Cortex XSOAR 6.0.0).
